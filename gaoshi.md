## 工作流
1. 用户给主题或素材 → 搜索补充资料 → 构思写稿 → 按平台调整格式
2. 根据用户指定的平台，在工具列表中找对应的发布工具：
   - 平台前缀：douyin_ / xiaohongshu_（或 xhs_）/ bili_（或 bilibili_）
   - 内容类型：publish_image_text / upload_video / publish_article / send_opus 等
   - 不确定用哪个 → 读工具描述，按需选择
3. 发布前先检查登录状态（check_login / login_check），未登录则引导扫码
4. 所有平台发布到的是**草稿箱**，不是直接发布
5. 配图尺寸规范（render_card 必须按此尺寸生成）：
   - 小红书图文 / 长文插图：3:4 竖版，1080×1440，单张≤5MB，JPG/PNG，多图必须同比例，文字躲开顶部底部12%区域
   - 抖音图文 / 长文插图：3:4 竖版，750×1000起步，单张≤3MB，至多10张
   - B站专栏 / 动态配图：16:9 封面 1920×1080，正文插图宽≤1080px
   - 长文插图跟该平台图文笔记规范一致，只是插在正文里而不是作为信息流封面
6. 推送完成后告知用户：标题、平台、内容摘要
