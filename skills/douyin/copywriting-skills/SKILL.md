# Copywriting Skills (文案技巧)

## Overview

Douyin copywriting is fundamentally different from other platforms — it's oral, conversational, and fast-paced. Users skim while scrolling video feeds. Sentences are short. Tone is casual. Value is delivered in the first 15 characters of every paragraph. The writing style mirrors how people actually talk, not how they write essays.

## Core Principles

1. **口语化 (Oral/Casual)**: Write like you talk. Short sentences. Colloquial phrases. Natural pauses.

2. **强钩子 (Strong Hook)**: First line must grab attention. No warm-up, no "最近如何如何", no intro.

3. **快节奏 (Fast Pacing)**: No long paragraphs. Every sentence earns its place. Cut filler words ruthlessly.

4. **互动感 (Interactive Feel)**: "你猜怎么着" / "试试看" / "你同意吗" — engage the reader directly.

## Platform Specs

| Content Type | Body Limit |
|---|---|
| Video Description | ≤500 characters |
| Image-Text Body | ≤1000 characters |
| Article Body | 300-8000 characters |

## Writing Patterns

### Video Description (≤500 chars)

```
HOOK (first 15-20 chars visible in feed)
↓
Key takeaway or teaser (1-2 sentences)
↓
CTA: Like/Follow/Comment
↓
#话题 #标签
```

Example:
```
这个方法改变了我做XX的方式🔥

3个步骤，5分钟搞定。最重要的是第3步，很多人不知道。

收藏⭐明天试试，评论区告诉我效果如何

#效率工具 #干货分享
```

### Image-Text Body (≤1000 chars)

```
HOOK IMAGE + arresting first line
↓
Point 1: brief explanation
↓
Point 2: brief explanation
↓
Point 3: brief explanation
↓
Summary + CTA
```

### Article (300-8000 chars)

```
OPENING HOOK (100 chars): problem statement or bold claim
↓
Section 1: background/context
Section 2: main content/tutorial
Section 3: deep dive/examples
Section 4: counterpoints/limitations
↓
CONCLUSION: key takeaways + CTA
```

## Voice and Tone

| Element | Approach |
|---|---|
| Sentence length | 5-25 characters, mix short and medium |
| Vocabulary | Everyday words, minimal jargon |
| Rhythm | Short-long-short pattern |
| Emotion | Direct, genuine, not performative |
| Humor | Natural, self-aware, not forced |

## Common Mistakes

| Mistake | Fix |
|---|---|
| Formal/essay tone | Write like you're talking to a friend |
| Long intro paragraph | Start with the most interesting line |
| Keyword stuffing | Natural use of keywords only |
| No CTA | Always end with one clear action |
| Overthinking | Done > perfect for Douyin |
