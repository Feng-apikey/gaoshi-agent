# Image-Text Creation (图文创作)

## Overview

Douyin image-text posts are swipeable carousel content that appears in the main feed alongside videos. Unlike Xiaohongshu's detailed carousels, Douyin image-text is shorter, faster-paced, and competes with video content for attention. The first image must be arresting enough to stop a video-consumed audience.

## When to Use

**Use when**: Creating image-text carousel posts, designing visual guides, product showcases, before-after comparisons, quick educational content.
**Do NOT use when**: Creating long-form articles (use article-writing), video content (use short-video-creation).

## Douyin Specs

| Spec | Value |
|---|---|
| **Title** | ≤55 characters |
| **Body** | ≤1000 characters |
| **Images** | 1-16 images (3:4 recommended) |
| **Tags** | #话题名, 3-5 tags recommended |

## Core Pattern

**Image 1**: Strong hook — arresting visual + bold headline (3-5 words)
**Images 2-6**: Content flow — each image advances the story
**Final image**: CTA — "关注看更多" / "收藏下次用"

## Implementation

1. **Hook Image**: Use high contrast, warm colors, clear subject. Add text overlay (bold, large font).
2. **Story Flow**: Each image = one point. Don't cram multiple ideas into one image.
3. **Text**: Keep scannable. 3-5 bullet points max per image. 24pt minimum font size.
4. **Visual Consistency**: Same 2-3 colors, same 1-2 fonts throughout carousel.
5. **CTA**: One clear action on final image.

## Douyin-Specific Tips

- Images compete with videos — thumbnails must be video-level engaging
- Hot topic tags (#热门话题) boost discovery but need real relevance
- First 3 seconds equivalent = first image impact
- Carousels with 4-7 images perform best
- Music background can be added to image-text posts

## Common Mistakes

| Mistake | Fix |
|---|---|
| Wall of text on one image | Spread across carousel, 3-5 bullets each |
| No hook in first image | Invest 50% effort in cover image |
| Too many images (>10) | Limit to 3-7, quality over quantity |
| Text too small | Test on phone, minimum 24pt |
| Generic CTA | Be specific: "评论区告诉我" beats "点赞关注" |
