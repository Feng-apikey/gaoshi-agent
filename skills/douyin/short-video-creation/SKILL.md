# Short Video Creation (短视频制作)

## Overview

Short video production for Douyin focuses on 7-60 second vertical content optimized for the platform's fast-paced, algorithm-driven feed. The core principle: the first 3 seconds determine everything. Douyin's audience has the fastest swipe behavior of any platform — content must hook instantly, deliver value rapidly, and end with a memorable moment.

**Key insight**: Douyin's algorithm heavily weights completion rate and replays. Well-produced short videos (15-60s, vertical 9:16) with strong hooks achieve 5-10x higher completion rates than unedited raw footage.

## When to Use

**Use when**: Creating vertical video for Douyin, filming vlogs/tutorials/reviews, editing short-form content under 60s, optimizing hooks for retention, creating product showcases, building personal brand through video.

**Do NOT use when**: Creating horizontal B站-style long videos, just needing image editing, live streaming content, cinematic/professional film production.

## Core Pattern

**Before** (unoptimized):
- 3+ minutes, viewers scroll away
- No hook, slow intro
- Poor audio, can't hear clearly
- No clear structure, rambles
- Weak or missing call-to-action

**After** (engaging):
- 15-60s, respects viewer time
- Captivating first 3 seconds
- Clear audio, professional quality
- Hook-value-CTA structure
- Clear CTA: like, follow, comment

## Implementation

### 1. Hook Design (First 3 Seconds)

The hook determines 80% of video success. Common patterns:
- **Curiosity**: "这个方法真的让我惊呆了..."
- **Specific promise**: "5个新手必学的技巧"
- **Problem-solution**: "别再犯这个错误了"
- **Immediate result**: Show before-after transformation instantly
- **Bold statement**: "这是我用过最好的平价替代"
- **Question**: "你知道为什么你的视频没人看吗？"

### 2. Content Structure

**15s video**: Hook(0-2s) → Value(2-13s) → CTA(13-15s)
**30s video**: Hook(0-3s) → Content(3-25s) → CTA(25-30s)
**60s video**: Hook(0-3s) → Content(3-50s) → Summary(50-55s) → CTA(55-60s)

### 3. Editing

Use JianYing (剪映) or CapCut. Key techniques:
- Jump cuts to maintain energy
- B-roll overlay for visual interest
- Auto-captions always on (60%+ watch without sound)
- Speed ramping: speed up boring parts, slow down reveals
- Text overlays for key points
- Background music at 10-20% volume

### 4. Audio

- Use external mic when possible (lavalier ¥50-300)
- Record in quiet space, close windows
- Douyin music library can be used for trendy sounds
- Voiceover must be clear and well-paced

### 5. CTA

End with one clear action: "关注我,每天分享干货" / "点赞收藏,下次容易找到" / "评论区告诉我你的看法"

## Douyin-Specific Tips

- First frame should show result or hook immediately — not "Hi everyone"
- Trending music/audio boosts discovery
- #话题标签: use 1-2 targeted mid-size tags, avoid oversaturated big tags with low account weight
- Post 7-10 PM for best engagement
- Publish consistently 3-7x per week

## Common Mistakes

| Mistake | Fix |
|---|---|
| Too long (>60s) | Cut to 15-45s |
| Weak or missing hook | Invest in first 3 seconds |
| No captions | Always add captions |
| Slow pacing | Fast cuts, remove pauses |
| Buried lead | Deliver value in first 5 seconds |
| Weak CTA | Clear, specific call-to-action |
